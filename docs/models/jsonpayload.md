# JsonPayload

Result of the task if the call uses an output schema


## Supported Types

### `{ [k: string]: any }`

```typescript
const value: { [k: string]: any } = {
  "sum": 4,
};
```

### `any[]`

```typescript
const value: any[] = [
  {
    "sum": 4,
  },
];
```

### `any`

```typescript
const value: any = {
  "sum": 4,
};
```
